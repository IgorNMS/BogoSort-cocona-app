## Bogo sort app
- um app simples para ordenar um conjunto de numeros da forma mais ineficiente possivel.
- o app foi criado usando o cocona, um pacote nuget que auxilia e facilita a criação de aplicações de console

## Como usar?
- .\BogoSort.exe sort --dalay 0 --announce Y/y ~~ N/n --numbers 9,898,234
- --dalay 0 configura o dalay em Ms entre uma interação e outra
- --announce Y/y(yes) N/n(no) configura se quer que cada interação seja logada no console
- --numbers 9,898,234 são os numeros que quer ordenar, so aceita numeros inteiros, a virgula serve como separação dos numeros
